Twig

Country extensions

This extension provides a set of Twig filters to work with countries.

Filters

country_icon

Returns the country icon from flag-icons.

{{ country|country_icon }}

country_icon_with_name

Returns the country icon from flag-icons with the country name.

{{ country|country_icon_with_name }}
